## üéÆ How to Play

1.  **Start the Game:** Open the link or the `index.html` file.
2.  **Controls:**
    * **Desktop:** Press `Spacebar` or `Click` the mouse to jump.
    * **Mobile:** Tap the screen to jump.
3.  **Objective:** Navigate the bird through the gaps in the pipes.
4.  **Game Over:** If you hit a pipe or the ground, the game ends. Refresh to try again!
